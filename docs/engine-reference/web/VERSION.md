|:-----|:-----|:-----|:---------|

## 知识风险评估

- **PixiJS v8**：与 v7 有较大的 API 变化（Container 替代 DisplayObject、新的 Assets 加载器等），需要参考最新文档
- **Vite v7**：相比 v6 有 Environment API 等新特性，但基础用法变化不大
- **TypeScript 5.7+**：稳定成熟，无需特别关注
