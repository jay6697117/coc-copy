# 渲染子系统

## 技术方案：PixiJS v8

### 核心概念
- **WebGL/WebGPU 自动检测**：`autoDetectRenderer()` 自动选择最佳渲染后端
- **Container 树**：所有可视对象组成树形结构
- **Sprite**：最基础的图像显示单元
- **Graphics**：矢量图形绘制
- **纹理图集（Spritesheet）**：多张小图合并为一张大图，减少 draw call

### COC 克隆特定需求
1. **等轴/俯视网格渲染** — 基地地图的网格系统
2. **图层管理** — 地面层、建筑层、单位层、UI 层分离
3. **相机控制** — 平移、缩放查看基地
4. **精灵动画** — 建筑动画、单位行走/攻击动画

### 性能关注点
- 大量建筑 + 围墙 + 装饰物的同时渲染
- 战斗场景中大量单位 + 弹道的渲染
- 使用 `ParticleContainer` 优化大量相似对象
