# 导航/寻路子系统

## 技术方案：自定义 A* 寻路

### COC 克隆特定需求
战斗系统中士兵的 AI 寻路是核心难点：

1. **A* 寻路算法** — 基于网格的路径查找
2. **目标选择 AI** — 不同兵种有不同的攻击偏好
   - 弓箭手：攻击最近单位
   - 巨人：优先攻击防御建筑
   - 哥布林：优先攻击资源建筑
3. **动态障碍物** — 围墙被摧毁后路径重新计算
4. **群体移动** — 大量单位同时移动的碰撞避免

### 推荐实现
- 使用基于网格的简化 A* 算法
- 建筑和围墙标记为不可通行
- 每个兵种定义 `targetPriority` 列表
- 战斗中定期重新计算路径（围墙被破坏时）

### 性能注意
- 批量寻路时做分帧处理，避免卡顿
- 缓存常用路径
- 考虑使用 Web Worker 做寻路计算
