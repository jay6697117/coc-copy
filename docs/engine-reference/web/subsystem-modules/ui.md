# UI 子系统

## 技术方案：PixiJS UI + HTML 混合

### 方案选择
COC 的 UI 复杂度较高（弹窗、列表、按钮等），推荐：
- **游戏内 HUD**：使用 PixiJS 直接渲染（资源面板、快捷操作）
- **复杂弹窗/表单**：使用 HTML/CSS 覆盖层（建筑详情、部落管理、设置）

### COC 克隆特定需求
1. **资源面板** — 顶部常驻显示金币/圣水/宝石数量
2. **建造菜单** — 底部建筑选择面板，带分类和滚动
3. **建筑详情弹窗** — 点击建筑后的升级/信息面板
4. **战斗部署栏** — 底部兵种选择栏
5. **战斗结果面板** — 星级评价 + 掠夺资源展示

### 推荐库
- `@pixi/ui`：PixiJS 官方 UI 组件库（按钮、滚动视图等）
