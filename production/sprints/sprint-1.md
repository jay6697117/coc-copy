# Sprint 1 — 基础设施与核心地基

- **时间范围**：灵活（预估 1-2 周）
- **目标**：搭建项目基础架构 + 实现网格地图和建筑放置的最小可交互原型
- **完成率**：[14/14] 任务完成 ✅

## 冲刺目标

> 这个冲刺结束时，玩家应该能在浏览器中看到一个网格地图，并能够在上面放置和移动建筑。

### 交付物
1. ✅ 可运行的 Vite + TypeScript + PixiJS 项目
2. ✅ 基础架构（事件总线、配置数据、游戏时钟、数据存储）
3. ✅ 网格地图渲染 + 相机控制（平移/缩放）
4. ✅ 建筑放置和移动的基础交互
