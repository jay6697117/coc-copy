# Sprint 2 — 资源经济与升级系统

- **时间范围**：灵活（预估 1-2 周）
- **目标**：让基地"活"起来 — 资源自动产出、建筑可升级、费用和时间体系完整
- **完成率**：[12/12] 任务完成 ✅

## 冲刺目标

> 这个冲刺结束时，金矿和圣水收集器会自动产出资源，建筑可以消耗资源升级，
> 升级有倒计时和等级效果提升，界面上能看到资源实时变化和建筑信息。

### 交付物
1. ✅ 资源管理系统（产出、存储上限、消耗、实时 UI 更新）
2. ✅ 升级系统（检查前置条件、扣费、倒计时、等级效果生效）
3. ✅ 建筑信息弹窗 UI（等级、属性、升级按钮）
4. ✅ 建筑建造扣费逻辑（放置新建筑时消耗对应资源）
