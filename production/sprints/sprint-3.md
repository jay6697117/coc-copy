# Sprint 3 — 军队系统与战斗引擎

- **时间范围**：灵活（预估 2-3 周）
- **目标**：实现兵种训练、军队编成和基础战斗循环（部署→战斗→结算）
- **完成率**：[12/12] 任务完成 ✅

## 冲刺目标

> 这个冲刺结束时，玩家可以建造兵营训练士兵，进入战斗场景部署兵力，
> 观看士兵自动攻击敌方建筑和防御塔自动还击，战斗结束后获得资源奖励。

### 交付物
1. ✅ 兵种配置数据（5 种 MVP 兵种）
2. ✅ 军队系统（训练队列、容量管理、兵种面板）
3. ✅ 战斗引擎（部署阶段→战斗循环→结算判定）
4. ✅ 战斗 AI（单位寻路、目标选择）
5. ✅ 防御 AI（防御塔自动瞄准）
6. ✅ 战斗场景渲染和结算 UI

## 技术风险
- 战斗循环的性能优化（大量单位同时移动和攻击）
- 寻路算法的简化版本（先用直线移动 + 障碍绕行，不用 A*）
- 战斗场景与主村庄场景的切换管理
